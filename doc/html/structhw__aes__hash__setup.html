Detailed Description

AES/Hash setup structure.

Field Documentation

hw_aes_direction hw_aes_hash_setup::aesDirection

AES direction. Only used when the mode is an AES mode.

uint32 hw_aes_hash_setup::aesIvCtrblk_0_31

In CBC mode IV[31:0] and in CTR mode the initial value of the 32 bits counter. Only used when the mode is an AES CBC/CTR mode.

uint32 hw_aes_hash_setup::aesIvCtrblk_32_63

In CBC mode IV[63:32] and in CTR[63:32]. Only used when the mode is an AES CBC/CTR mode.

uint32 hw_aes_hash_setup::aesIvCtrblk_64_95

In CBC mode IV[95:64] and in CTR[95:64]. Only used when the mode is an AES CBC/CTR mode.

uint32 hw_aes_hash_setup::aesIvCtrblk_96_127

In CBC mode IV[127:96] and in CTR[127:96]. Only used when the mode is an AES CBC/CTR mode.

bool hw_aes_hash_setup::aesKeyExpand

When true the key expansion process is execute. When false the key expansion process is not executed. The user should write the AES keys in CRYPTO_RAM. Only used when the mode is an AES mode.

uint32 hw_aes_hash_setup::aesKeys

The start address of the buffer containing the AES key.

hw_aes_key_size hw_aes_hash_setup::aesKeySize

AES key size. Only used when the mode is an AES mode.

bool hw_aes_hash_setup::aesWriteBackAll

When true all the AES resulting data is written to memory. When false only the final block of the AES resulting data is written to memory. Only used when the mode is an AES mode.

hw_aes_hash_cb hw_aes_hash_setup::callback

The callback function that is called when enable interrupt is true.

uint32 hw_aes_hash_setup::dataSize

The number of bytes that need to be processed. If this number is not a multiple of a block size, the data is automatically extended with zeros.

uint32 hw_aes_hash_setup::destinationAddress

The physical address (RAM only) where the resulting data needs to be written. If NULL the register is not written.

bool hw_aes_hash_setup::enableInterrupt

When true the callback function is called after the operation has ended.

uint8 hw_aes_hash_setup::hashOutLength

The number of bytes of the hash result to be saved to memory. Only used when mode is a Hash mode.

hw_aes_hash_mode hw_aes_hash_setup::mode

AES/Hash mode.

bool hw_aes_hash_setup::moreDataToCome

When false this is the last data block. When true more data is to come.

uint32 hw_aes_hash_setup::sourceAddress

The physical address of the input data that needs to be processed.
