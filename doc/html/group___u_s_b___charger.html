__STATIC_INLINE void hw_charger_configure ( void  )

Configures the Charger how to charge the battery.

The default configuration is:

  • Charger status: disabled
  • NTC : user defined
  • NTC_LOW : stop at 7/8 VDD
  • Die temperature protection level : 80C
  • Die temperature protection : enabled
  • Charge current : user defined
  • Charge voltage 2 x NiMh : 3.40V Li-phosphate : 3.60V LiCo / LiMn / NMC : 4.20V 3 x NiMh : 4.90V Custom : user defined
__STATIC_INLINE bool hw_charger_end_of_charge ( void  )

Check end-of-charge (Li-ion).

Returns
bool True if end-of-charge, else false.
Warning
It returns true even when the USB cable is attached. Thus, in order to reach to a safe conclusion about the battery charge status, the USB state (attached / detached) must also be taken into account.
