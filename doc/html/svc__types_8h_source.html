157  /*
158  * ISO/IEEE Std. 11073-20601™-2008 defines SFLOAT-Type data type which consist of 12-bit
159  * mantissa and 4-bit exponent in below order:
160  * Exponent Mantissa
161  * 4 bit 12 bit
162  */
163 
164  /* maximum exponent = 7
165  * minimum exponent = -8 (signed 4 bits)
