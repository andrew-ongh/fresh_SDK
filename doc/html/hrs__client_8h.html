typedef void(* hrs_client_heart_rate_measurement_notif_cb_t) (ble_client_t *client, hrs_client_measurement_t *measurement)

Heart Rate Measurement notification callback.

Called when Client received Heart Rate Measurement notification from server. In this callback OS_FREE(measurement) must be done to prevent leaking of memory caused by OS_MALLOC for measurement with variable length of rr values while data receiving.
