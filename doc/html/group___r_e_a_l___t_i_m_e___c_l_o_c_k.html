uint64_t rtc_get ( void  )

Update Real Time Clock value.

Returns
The current RTC time. This is expressed in ticks of the clock that clocks TIMER1 (e.g. XTAL32K). For example, if XTAL32K drives TIMER1, each RTC tick will be 1000000 / 32768 = 30.5uS
Warning
This function is called only from OS Tasks.
