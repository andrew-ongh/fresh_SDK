static void hw_timer0_set_t0_reload ( uint16_t  m_value,
uint16_t  n_value 
)
inlinestatic

Set reload value for T0-counter.

T0 counter value is decremented on each clock cycle. At the beginning it's loaded from M-register and then, once reached zero, loaded from N-register (and then M and N again).

PWM0 is high when counting down M-register and low when counting down N-register. For PWM1 is the opposite.

Parameters
[in]m_valueM-register reload value
[in]n_valueN-register reload value
