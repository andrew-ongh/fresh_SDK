inlinestatic

Configure event counting and triggering state for whole GPIO port.

In enabled and state bitmasks each bit describes state of corresponding pin in port. For enabled 0 means disabled and 1 means enabled. For state 0 means event is triggered on low state and 1 means trigger is on high state.

void hw_wkup_register_interrupt ( hw_wkup_interrupt_cb  cb,
uint32_t  prio 
)

Register interrupt handler.

Callback function is called when interrupt is generated, i.e. event counter reaches configured value. Interrupt is automatically enabled after calling this function. Application should reset interrupt in callback function using hw_wkup_reset_interrupt(). If no callback is specified, interrupt will be automatically cleared by the driver.

Parameters
[in]cbcallback function
[in]priothe priority of the interrupt
