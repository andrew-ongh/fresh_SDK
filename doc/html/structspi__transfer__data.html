Detailed Description

Transfer structure for complex SPI transaction.

One of wbuf and rbuf can be NULL. If both wbuf and rbuf are not NULL, duplex transmission is performed. Buffer addresses and lengths must be be SPI-word-aligned (no alignment needed for 9-bit SPI word configurations).
