bool security_get_unique_device_id ( uint8_t *  udi)

Get unique device identifier.

Function reads two fields from OTP (position/package and tester/timestamp) and joins them into unique device identifier.

Note
udi must point to the buffer which has at least UNIQUE_DEVICE_ID_LEN bytes in length.
Parameters
[out]udiunique device identifier buffer
Returns
true if unique device identifier has been read properly, false otherwise
Warning
Function uses OTP. hw_otpc_init() and the hw_otpc_set_speed() functions must be called before using this function.
