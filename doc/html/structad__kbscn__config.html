uint8_t ad_kbscn_config::debounce_press_time

Debounce time for button press

This value is expressed as number of keyboard full scan cycles. Each full scan cycle takes num_rows * (row_scan_time + 2) clock cycles.

uint8_t ad_kbscn_config::debounce_release_time

Debounce time for button release

This value is expressed as number of keyboard full scan cycles. Each full scan cycle takes num_rows * (row_scan_time + 2) clock cycles.

uint8_t ad_kbscn_config::inactive_time

Inactive time to wait after last kbd event before allowing system to sleep.

This value is expressed as number of keyboard full scan cycles. Each full scan cycle takes num_rows * (row_scan_time + 2) clock cycles.
