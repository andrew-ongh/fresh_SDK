struct i2c_device_config_t* i2c_bus_dynamic_data::current_device

This keeps track of last device that was used on I2C bus. When device changes all controller parameters must be also changed. If CONFIG_I2C_ONE_DEVICE_ON_BUS is defined as 1 there is no need to check/update controller settings at every transfer so this field is not needed
