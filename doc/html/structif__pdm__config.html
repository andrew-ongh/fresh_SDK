bool if_pdm_config::bypass_in_filter

This is used to bypass the sample rate converter in upsampling filters.

The mode of the PDM interface. Master or slave.

uint8_t if_pdm_config::pdm_div

This is used to set the PDM_DIV field of the PDM_DIV_REG. If this equals 0 the PDM_DIV is set to 8.

IF_PDM_SRC_DIRECTION if_pdm_config::src_direction

The direction of the sample rate converter. For example, when direction is IF_PDM_DIRECTION_IN and src_direction is IF_PDM_SRC_DIRECTION_PCM the PDM signal will be passed to the PCM interface. Or when the PDM direction is out and src_direction is IF_PDM_SRC_DIRECTION_REG the SRC I/O registers are used for PDM input.

unsigned int if_pdm_config::src_sample_rate

The sample rate of the sample rate converter (Hz). This is only used when src_direction is IF_PDM_SRC_DIRECTION_REG.
