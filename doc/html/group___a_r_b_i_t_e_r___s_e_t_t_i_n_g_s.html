#define dg_configCOEX_ENABLE_CONFIG   (0)

Custom arbiter configuration support When defined, coex is configurable and priorities can be set: -either manually, per mac, using coex api -or automatically, by the PTIs provided by each mac.

When not defined, coex operates with the default/fixed priority scheme: BLE traffic has always higher priority than FTDF.
