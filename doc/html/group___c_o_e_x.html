enum HW_COEX_DIAG_MODE

Diagnostics mode.

Arbiter diagnostics use the BLE diagnostics bus. BLE diagnostics are assigned to the following GPIOs:
ble_diag_0: P2_0
ble_diag_1: P2_1
ble_diag_2: P2_2
ble_diag_3: P1_0
ble_diag_4: P1_1
ble_diag_5: P1_2
ble_diag_6: P1_3
ble_diag_7: P2_3

Enumerator
HW_COEX_DIAG_MODE_1 

Use bits [5:3] of the BLE diagnostics bus.
bit 5 : Closing pulse
bits [4:3] : Decision bits as per HW_COEX_MAC_TYPE

HW_COEX_DIAG_MODE_2 

Use bits [7:3] of the BLE diagnostics bus.
bit 7 : FTDF TX/RX enable bit 6 : BLE TX/RX enable bit 5 : Closing pulse
bits [4:3] : Decision bits as per HW_COEX_MAC_TYPE

HW_COEX_DIAG_MODE_3 

Use bits [7:2] of the BLE diagnostics bus.
bit 7 : Always 0
bits [6:3] : Decision Pointer (corresponding to the PTI table), that is the registers COEX_PRI1_REG to COEX_PRI15_REG
bit 2 : Closing pulse OR RADIO BUSY
