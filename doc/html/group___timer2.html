static void hw_timer2_set_hw_pause ( bool  enable)
inlinestatic

Enable/disable hardware pause for PWM generation.

Calling this will enable or disable hardware pause during PWM generations.

When hardware pause is enabled PWM counter is stopped during RF activity and all PWM outputs are set to 0 for RF duration. This can suppress the PWM switching noise.

When hardware pause is disabled timer2 will not be stopped during RF transmission, PWM will operate normally. This can introduce PWM switching noise.

static void hw_timer2_set_sw_pause ( bool  enable)
inlinestatic

Enable/disable software pause for PWM generation.

When software pause is enabled then output PWM pins are set to 0. This allows to reduce PWM switching noise.

When software pause is disabled then timer 2 is unblocked from software pause. PWM can still be paused if hardware pause is enabled.
