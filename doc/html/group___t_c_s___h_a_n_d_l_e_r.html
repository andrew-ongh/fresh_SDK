Function Documentation

bool sys_tcs_store_pair ( uint32_t  address,
uint32_t  value 
)

Store TCS <address, value> pair in the Global TCS array if it points to a register that is not in the AON power dowmain or is not retained, else it applies the vaule to the register.

Parameters
[in]addressthe address of the register
[in]valuethe value for this register
Returns
true if the chip is calibrated (the BANDGAP setting has been applied), else false.
Warning
When this function is called, the RC16 must have been set as the system clock!

Variable Documentation

bool sys_tcs_is_calibrated_chip

Flag to check whether the TCS is written or not.

true, the TCS is written (the BANDGAP_REG value exists in the TCS) false, the TCS is empty

uint16_t sys_tcs_xtal16m_settling_time

The XTAL16M settling time (in TCS).

XTAL32K, the settling time is expressed in clock cycles RCX, not applicable; the SDK hard-coded value is used If it is zero, the hard-coded value (dg_configXTAL16_SETTLE_TIME) is applied.
