void ad_gpadc_acquire ( void  )

Acquire access to GPADC.

This function waits for GPADC to become available, and locks it. This function can be called several times, but number of ad_gpadc_release() calls must match number of calls to this function.

This function should be used if normal ad_gpadc_read() is not enough and some other GPADC controller calls are required. In this case typical usage for this function would look like this:

ad_gpadc_acquire(); ... ad_gpadc_release();

bool ad_gpadc_acquire_to ( uint32_t  timeout)

Attempt to acquire exclusive access on GPADC within a timeout period.

This function waits for GPADC to become available until the timeout expires. When GPADC is acquired the resource is locked and requires an ad_gpadc_release() call to unlock it.

Parameters
[in]timeoutnumber of ticks to wait 0 - no wait take GPADC if it is available RES_WAIT_FOREVER - wait until GPADC becomes available Other value specifies how many ticks to wait until GPADC becomes available
Returns
true if GPADC has been acquired, false otherwise
uint16_t ad_gpadc_get_source_max ( gpadc_source  source)

Return maximum value that can be read for ADC source.

Value returned by ad_gpadc_read() can have 10 to 16 bits (right aligned) depending on oversampling specified in source description. This function will return value 0x3FF, 0x7FF, 0xFFF, 0x1FFF, 0x3FFF, 0x7FFF or 0xFFFF depending on oversampling.

Parameters
[in]sourcehandle to GPADC source
Returns
maximum value that can be returned by ad_gpadc_read

gpadc_source ad_gpadc_open ( const gpadc_source_id  id)

Open source connected to GPADC.

This function will set up configuration parameters to GPADC controller. It has to be called before ad_gpadc_acquire() to get a handle to the source like this:

source_handle = ad_gpadc_open(GPADC_SOURCE); ad_gpadc_acquire(); ... ad_gpadc_release(); ad_gpadc_close(source_handle);

Parameters
[in]idid of the source connected to GPADC (name should match entries defined by GPADC_SOURCE macro)
Returns
source handle that can be used with other functions
